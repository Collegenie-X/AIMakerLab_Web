### Tesla의 핵심 자산

> **[마인드맵 다이어그램 생략]** 주요 개념은 본문에서 설명합니다.

### 실전 알고리즘: Occupancy Network

**목적**: 카메라 8개 입력 → 3D 공간 점유 예측 → 경로 계획

**알고리즘 흐름**

> **[간단한 흐름도]** 본문에서 단계별로 설명합니다.

**핵심 수식**
```
점유 확률 P(occupied | x, y, z) = σ(CNN(I₁...I₈, x, y, z))

여기서:
- I₁...I₈: 8개 카메라 이미지
- (x, y, z): 3D 공간 좌표
- σ: Sigmoid 함수
- CNN: 학습된 신경망 (5억 파라미터)
```
