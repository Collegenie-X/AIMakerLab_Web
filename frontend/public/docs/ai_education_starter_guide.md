# Part 2: 단계별 학습 로드맵

## 2.1 1단계: 블록코딩 (초등 1-3학년)

### 학습 목표
- 프로그래밍 기본 개념 (순차, 반복, 조건)
- 논리적 사고력 향상
- 간단한 게임/애니메이션 제작

### 커리큘럼 (12주)

```mermaid
gantt
    title 블록코딩 12주 커리큘럼
    dateFormat YYYY-MM-DD
    
    section 기초 (1-4주)
    Scratch 기본            :a1, 2025-01-01, 7d
    캐릭터 움직이기         :a2, after a1, 7d
    소리와 색상 제어        :a3, after a2, 7d
    간단한 애니메이션       :a4, after a3, 7d
    
    section 중급 (5-8주)
    조건문 (만약~라면)      :b1, after a4, 7d
    반복문 (계속 반복하기)  :b2, after b1, 7d
    변수 사용하기           :b3, after b2, 7d
    미니 게임 만들기        :b4, after b3, 7d
    
    section 프로젝트 (9-12주)
    AI 게임 기획            :c1, after b4, 7d
    게임 제작               :c2, after c1, 14d
    발표 준비               :c3, after c2, 7d
```

### 프로젝트 예시: 고양이 피하기 게임

```
게임 규칙:
1. 화살표 키로 캐릭터 이동
2. 떨어지는 장애물 피하기
3. 10초 버티면 승리!

필요한 개념:
- 키보드 입력 감지
- 랜덤 위치 생성
- 충돌 감지
- 타이머 변수
```

- 라즈베리파이

```mermaid
graph LR
    A[꾸준함] --> D[성공]
    B[실전 프로젝트] --> D
    C[공유와 협업] --> D
    
    A --> A1[하루 30분]
    B --> B1[완성도 70%도 OK]
    C --> C1[GitHub 업로드]
    
    style D fill:#ffd700,color:#000
```

2. **실전**: 완벽하지 않아도 일단 만들어보기
