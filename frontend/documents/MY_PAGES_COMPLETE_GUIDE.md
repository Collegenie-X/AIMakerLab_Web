# 나의 페이지 완전 가이드 (개선된 UI & 이모티콘)

## 🎨 개요

일반 사용자를 위한 **4개의 마이 페이지**가 완성되었습니다!
모든 페이지는 **이모티콘과 개선된 UI**로 쇼핑몰/Udemy 스타일을 따릅니다.

## 📱 메뉴 구조

### 데스크톱 헤더 드롭다운
```
[사용자 이메일 ▼]
├─ 📚 나의 강의
├─ 📝 나의 문의
├─ 🎨 나의 갤러리
├─ (구분선)
├─ ⚙️ 프로필 설정
├─ (구분선)
└─ 🚪 로그아웃
```

### 모바일 메뉴
```
☰ 햄버거 메뉴
├─ 📧 [사용자 이메일]
├─ [로그아웃 버튼]
├─ (구분선)
└─ 📚 내 학습
    ├─ 📚 나의 강의
    ├─ 📝 나의 문의
    ├─ 🎨 나의 갤러리
    └─ ⚙️ 프로필 설정
```

## 📖 4개의 마이 페이지

### 1. 📚 나의 강의 (`/my-courses`)

**URL**: `/my-courses`

**기능**:
- ✅ 수강 중인 강의 목록 조회
- ✅ 강의 상태 표시 (진행중/예정/완료)
- ✅ 진행률 표시
- ✅ 강의 카드 클릭 → 강의 상세 페이지로 이동

**UI 특징**:
```
┌─────────────────────────────────┐
│ 📚 Header                       │
├─────────────────────────────────┤
│ [📘] 📚 나의 강의               │
│ 수강 중인 강의 목록입니다        │
│                                 │
│ ┌──────────┐ ┌──────────┐      │
│ │ 📘 AI    │ │ 🤖 아두  │      │
│ │ 김선생님  │ │ 이선생님  │      │
│ │ 진행률65%│ │ 진행률40%│      │
│ │ [강의보기]│ │ [강의보기]│      │
│ └──────────┘ └──────────┘      │
├─────────────────────────────────┤
│ Footer                          │
└─────────────────────────────────┘
```

**강의 카드 정보**:
- 상태 배지 (진행중🟢/예정🔵/완료⚪)
- 카테고리 태그
- 강의 제목
- 강사명
- 시작일/종료일 (📅)
- 진행률 바 (진행 중인 경우)
- [강의 보기] 버튼

**빈 상태**:
- 📚 큰 아이콘
- "수강 중인 강의가 없습니다"
- [강의 둘러보기] 버튼

### 2. 📝 나의 문의 (`/my-inquiries`)

**URL**: `/my-inquiries`

**기능**:
- ✅ 견적 문의 내역 조회
- ✅ 출강 수업 문의 내역 조회
- ✅ 답변 상태 확인
- ✅ 관리자 답변 확인

**UI 특징**:
```
┌─────────────────────────────────┐
│ Header                          │
├─────────────────────────────────┤
│ [📄] 📝 나의 문의               │
│ 견적 문의와 출강 문의 내역       │
│                                 │
│ ┌─────────────────────────────┐ │
│ │ 📦 견적문의 | 2024-11-15    │ │
│ │ AI 코딩 교구 세트 견적 문의  │ │
│ │ 제품: 아두이노 스타터 키트   │ │
│ │ 수량: 30개                   │ │
│ │                              │ │
│ │ 💬 문의 내용:                │ │
│ │ 초등학교 3학년 학생 30명...  │ │
│ │                              │ │
│ │ ✅ 답변완료                  │ │
│ │ 관리자 답변: 안녕하세요...   │ │
│ └─────────────────────────────┘ │
├─────────────────────────────────┤
│ Footer                          │
└─────────────────────────────────┘
```

**문의 카드 구성**:
- **타입 아이콘**: 
  - 📦 견적문의 (보라색)
  - 🚗 출강문의 (파란색)
- **상태 배지**:
  - ✅ 답변완료 (초록색)
  - ⏳ 진행중 (주황색)
  - 💬 답변대기 (파란색)
- **문의 정보**:
  - 제목
  - 제품명/수량 또는 지역/기간
  - 작성일 (📅)
- **문의 내용** (회색 배경)
- **관리자 답변** (상태별 색상 배경)

**빈 상태**:
- 📄 큰 아이콘
- "문의 내역이 없습니다"
- [제품 견적 문의] [출강 수업 문의] 버튼

### 3. 🎨 나의 갤러리 (`/my-gallery`)

**URL**: `/my-gallery`

**기능**:
- ✅ 업로드한 작품 조회
- ✅ 작성한 후기 조회
- ✅ 필터링 (전체/작품/후기)
- ✅ 조회수/좋아요 수 확인
- ✅ 상태 확인 (공개/비공개/검토중)

**UI 특징**:
```
┌─────────────────────────────────┐
│ Header                          │
├─────────────────────────────────┤
│ [🎨] 🎨 나의 갤러리             │
│ 내가 업로드한 작품과 후기        │
│                                 │
│ [🎨 전체 5] [🏆 작품 3] [⭐ 후기 2] │
│                                 │
│ ┌────┐ ┌────┐ ┌────┐          │
│ │🏆작품│ │🏆작품│ │⭐후기│          │
│ │이미지│ │이미지│ │이미지│          │
│ │제목  │ │제목  │ │제목  │          │
│ │설명  │ │설명  │ │설명  │          │
│ │👁125│ │👁89 │ │👁45 │          │
│ │❤️23 │ │❤️15 │ │❤️8  │          │
│ │✅공개│ │✅공개│ │⏳검토│          │
│ └────┘ └────┘ └────┘          │
├─────────────────────────────────┤
│ Footer                          │
└─────────────────────────────────┘
```

**필터 버튼**:
- 🎨 전체 (배지: 총 개수)
- 🏆 작품 (배지: 작품 개수)
- ⭐ 후기 (배지: 후기 개수)

**갤러리 카드**:
- **이미지** (호버 시 확대 효과)
- **오버레이 배지**:
  - 좌상단: 🏆 작품 / ⭐ 후기
  - 우상단: ✅ 공개 / 🔒 비공개 / ⏳ 검토중
- **제목** (1줄)
- **설명** (2줄)
- **통계**:
  - 👁️ 조회수
  - ❤️ 좋아요
  - 📅 작성일
- **[상세보기] 버튼**

**호버 효과**:
- 이미지 확대
- 어두운 오버레이
- 👁️ "자세히 보기" 텍스트

**빈 상태**:
- 🎨/🏆/⭐ 큰 아이콘 (필터별)
- "업로드한 항목이 없습니다"
- [작품 갤러리 보기] [수업 후기 보기] 버튼

## 🎨 UI 개선 포인트

### 1. **이모티콘 사용** ✨
```
📚 나의 강의
📝 나의 문의
🎨 나의 갤러리
⚙️ 프로필 설정
```

### 2. **컬러풀한 아이콘**
```
🔵 파란색 - 강의 (BookOpen)
🟢 초록색 - 문의 (FileText)
🟣 보라색 - 갤러리 (Images)
⚫ 회색 - 프로필 (User)
```

### 3. **상태별 이모티콘**
```
✅ 완료/공개
⏳ 진행중/검토중
💬 대기
🔒 비공개
```

### 4. **그라데이션 배경**
```css
bg-gradient-to-b from-gray-50 to-white
```

### 5. **호버 효과**
- 그림자 강화: `hover:shadow-xl`
- 이미지 확대: `group-hover:scale-110`
- 아이콘 확대: `group-hover:scale-110`

### 6. **카드 디자인**
- 둥근 모서리
- 부드러운 그림자
- 좌측 컬러 보더 (문의 카드)
- 트랜지션 효과

### 7. **빈 상태 디자인**
- 큰 원형 아이콘 배경
- 명확한 안내 문구
- CTA 버튼 제공

## 🎯 사용자 플로우

### 견적 문의 확인하기
```
1. 헤더에서 이메일 클릭
2. "📝 나의 문의" 선택
3. 문의 내역 확인
4. 답변이 있으면 초록색 박스로 표시됨
```

### 갤러리 확인하기
```
1. 헤더에서 이메일 클릭
2. "🎨 나의 갤러리" 선택
3. 필터 선택 (전체/작품/후기)
4. 카드 클릭 → 갤러리 상세 페이지
```

### 강의 수강하기
```
1. 헤더에서 이메일 클릭
2. "📚 나의 강의" 선택
3. 강의 카드에서 진행률 확인
4. [강의 보기] 클릭 → 강의 상세 페이지
```

### 프로필 수정하기
```
1. 헤더에서 이메일 클릭
2. "⚙️ 프로필 설정" 선택
3. "프로필 정보" 탭에서 정보 수정
4. [프로필 업데이트] 클릭
```
