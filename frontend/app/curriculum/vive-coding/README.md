# 바이브 코딩 프로그램

기획부터 실행까지, IoT 제작과 AI 서비스 개발 실전 프로젝트입니다.

## 과정 개요

### 📌 2개 트랙

#### 트랙 1: IoT 제작
- **내용**: 아두이노 + 라즈베리파이 + 레이저 커팅
- **결과물**: 완성도 높은 IoT 제품
- **특징**: 
  - 하드웨어 설계 및 회로 구성
  - 레이저 커팅기로 전문적인 외형 제작
  - 프로토타입부터 양산 준비까지

#### 트랙 2: 바이브 코딩 서비스 개발
- **내용**: ChatGPT API + Next.js + Flask
- **결과물**: 실제 배포 가능한 웹 서비스
- **도구**: Figma, v0, AI Studio, Cursor
- **특징**:
  - AI 도구를 실행자로 활용
  - 디버깅 방법 학습
  - 기획 능력 + AI 리터러시

### 🎯 학습 목표

❌ **코딩 수업이 아닙니다**  
✅ **실행자 + 디버깅 방법을 배웁니다**  
✅ **기획 능력을 키웁니다**  
✅ **AI 리터러시를 습득합니다**  

### 👥 대상
- 중학생 1학년 이상 (기초 코딩 경험 권장)
- 프로젝트 기획 및 실행에 관심 있는 학생
- 포트폴리오 제작을 원하는 학생

- 프로토타입 발표

- 전문 서비스 기획
- Next.js 프론트엔드
- Flask 백엔드 + ChatGPT API
- 통합 및 디버깅

## 사용 도구

### IoT 제작
- **하드웨어**: 아두이노, Raspberry Pi
- **센서**: 온도, 습도, 모터, 카메라 등
- **제작**: 레이저 커팅기, 3D 프린터
- **설계**: Fritzing, KiCad, Fusion 360

### 바이브 코딩
- **디자인**: Figma
- **AI 코딩**: v0, Cursor, AI Studio
- **프론트엔드**: Next.js 15, React
- **백엔드**: Flask, ChatGPT API
- **배포**: Vercel, Railway
