# ChatGPT AI 코딩 커리큘럼

## 📌 개요

**ChatGPT와 함께하는 AI 코딩** 프로그램은 AI를 단순한 도구가 아닌 **협업 파트너**로 활용하여 콘텐츠 제작부터 웹 서비스 개발까지 경험하는 실전 코딩 교육입니다.

## 🎯 핵심 철학

### 1. 열린 프롬프트 전략

> **ChatGPT는 이미 주니어 개발자 3년차 수준입니다.**  
> 제한적인 지시가 아닌, 상황과 문제를 정의하여 AI에게 최선의 방법을 물어보는 **협업적 접근**을 배웁니다.

**❌ 마이크로매니징 방식:**
```
"Python으로 만들어줘"
"HTML 파일 1개로 만들어줘"
```

**✅ 파트너 협업 방식:**
```
"저는 고등학생이고, 이런 문제가 있습니다.
어떤 방식으로 해결하는 것이 가장 좋을까요?
기술 스택과 배포 방법을 추천해주세요."
```

### 2. 역공부 방식

전통적 학습: `이론 → 실습 → 프로젝트` (3개월)  
역공부 방식: `프로젝트 → 질문 → 이해` (3일)

**4단계 역공부 프로세스:**
1. **결과물 생성** - AI가 먼저 만듦
2. **작동 확인** - 테스트
3. **질문 생성** - "왜? 어떻게?"
4. **깊이 이해** - 필요한 것만 배우기

### 3. 4대 역량

- 🧠 **기획자**: 문제 정의 + 알고리즘 설계
- ⚡ **실행자**: AI 협업 + 빠른 구현
- 🔧 **디버거**: 성찰 + 알고리즘 개선
- 🚀 **기업가**: 실사용자에게 가치 전달

## 📚 과정 구성

### 파트 1: AI 콘텐츠 크리에이터 (ChatGPT Playground)

**학습 내용:**
- 열린 프롬프트 작성법
- 페르소나 설정 및 활용
- 체이닝 프롬프트 전략
- 질문-답변 예측 기획

**프로젝트:**
- 동화책 만들기
- 교육 콘텐츠 제작
- 블로그 시리즈
- 카드뉴스, 퀴즈

### 파트 2: AI Agent 개발자 (ChatGPT API)

**학습 내용:**
- ChatGPT API 활용
- 웹 서비스 기획
- MVP 개발 방법론
- 테스트 및 배포
- 사용자 피드백 반영

**프로젝트:**
- 학습 도우미 챗봇
- 자동 알림 시스템
- Q&A 웹 서비스
- 실전 AI Agent

## 🎓 학년별 트랙

### 초등 (4-6학년): AI 콘텐츠 크리에이터

- **목표**: AI와 친해지고 창의적 콘텐츠 제작
- **기간**: 4주 (주 1회 2시간)
- **결과물**: 동화책 1개, 콘텐츠 3개

### 중등 (1-2학년): AI 협업 개발자

- **목표**: 열린 프롬프트 + 웹 서비스 개발
- **기간**: 8주 (주 1회 2시간)
- **결과물**: 콘텐츠 10개, 웹 서비스 1개

### 고등 (중3, 고1-2): AI Agent 서비스 개발자

- **목표**: 실사용자가 있는 서비스 운영
- **기간**: 12주 (주 1회 3시간)
- **결과물**: 실사용 웹 서비스 (50명+ 사용자)

## 💡 차별화 포인트

### 1. 학년별 전략적 차별화

| 구분 | 초·중학생 | 고등학생 |
|------|----------|---------|
| **관점** | 학습자 | 서비스 제공자 |
| **품질** | 작동하면 OK | 주니어 3년차 수준 |
| **증명** | "만들어봤어요" | "300명이 사용 중" |
| **도구** | 무료 위주 | 유료 적극 활용 |

### 2. 실전 배포 중심

**학습용 프로젝트 (초중):**
- 로컬에서 작동
- 나만 사용
- 과정이 중요

**실전 서비스 (고등):**
- 인터넷 접근 가능
- 실사용자 50명+
- 결과와 임팩트가 중요
- 포트폴리오 제출 가능

### 3. 성찰 중심 디버깅

단순 버그 수정이 아닌:
- 왜 이런 문제가 생겼을까? (성찰)
- 알고리즘이 잘못됐나? (분석)
- 더 나은 방법은? (개선)
- 배운 것 기록 (학습)
