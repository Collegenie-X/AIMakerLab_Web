## 📚 프로젝트 1: 인터랙티브 동화책 만들기 (3시간)

### 프로젝트 개요
독자가 선택에 따라 이야기가 달라지는 **인터랙티브 동화책** 제작

### 1단계: 아이디어 발굴 (30분)

**학생 활동:**
```
💭 ChatGPT와 브레인스토밍:

"초등학생이 좋아할 인터랙티브 동화 주제 10가지 제안해줘.
각 주제마다:
- 기본 설정
- 선택 포인트 3개
- 예상 독자 반응
을 포함해줘"
```

**ChatGPT 응답 예시:**
```
1. 마법 학교 입학 첫날
   - 설정: 11살 주인공이 마법 학교에 입학
   - 선택 포인트:
     * 어느 기숙사에 배정받을까?
     * 첫 친구는 누구?
     * 첫 수업 선택
   - 예상 반응: 해리포터 팬이 좋아함
...
```

**결과물:** 10개 아이디어 중 1개 선택

### 2단계: 기획서 작성 (40분)

**ChatGPT 활용:**
```
선택한 주제: "마법 학교 입학 첫날"

다음 형식으로 상세 기획서를 만들어줘:

## 동화 기획서
- 제목: 
- 대상 독자:
- 주인공 설정:
- 세계관:
- 주요 등장인물 (3명):
- 이야기 구조:
  * 시작
  * 선택 포인트 1
    - A 선택 시 전개
    - B 선택 시 전개
  * 선택 포인트 2
  * 결말 (3가지)
```

**학생 역할:**
- ChatGPT 결과물 검토
- 마음에 안 드는 부분 수정 요청
- 자신만의 아이디어 추가

**개선 예시:**
```
"주인공 나이를 13살로 바꾸고, 
배경을 한국 전통 마법으로 변경해줘.
도깨비, 구미호 같은 한국 요소를 넣어줘"
```

### 3단계: 프로토타입 제작 (60분)

**1장 작성:**
```
위 기획서를 바탕으로 1장을 작성해줘.
- 분량: 1000자
- 톤: 초등 고학년이 읽기 편한 문체
- 1장 끝에서 첫 번째 선택 포인트 제시
- 선택지 2개를 명확하게 표시
```

**2-3장 작성 (분기):**
```
독자가 [A 선택지]를 선택했다고 가정하고 2장을 써줘.
- 선택의 결과가 명확히 드러나게
- 새로운 선택 포인트 추가
```

**빠른 반복:**
- 각 장을 5-10분 만에 생성
- 마음에 안 들면 즉시 수정 요청
- 여러 버전 비교

### 4단계: 개선 및 완성 (50분)

**일관성 체크:**
```
지금까지 작성한 1-4장을 검토해줘:
- 캐릭터 성격이 일관된가?
- 복선이 잘 깔렸나?
- 개선할 점은?
```

**삽화 설명 생성:**
```
각 장의 핵심 장면을 이미지로 만들고 싶어.
DALL-E에 입력할 프롬프트를 장마다 작성해줘.
- 화풍: 디즈니 애니메이션 스타일
- 구도: 3인칭 시점
```

**최종 검토:**
- 친구에게 읽어보게 하기
- 피드백 받아 마지막 수정

### 결과물
✅ 4-5개 장으로 구성된 완성된 인터랙티브 동화  
✅ 최소 2개의 분기점과 3개의 엔딩  
✅ 삽화 프롬프트 5개  
✅ 제작 과정 문서

```
- 장소:

- 동작:
- 조명:
- 소품:
```

```
1. 제목 슬라이드
2. 콘셉트 소개
3. 핵심 장면 3개

5. 기대 효과

각 슬라이드에 들어갈 텍스트를 작성해줘
```

### 결과물
✅ 완성된 뮤직비디오 시나리오  
✅ 장면별 스토리보드 (10-15장면)  
✅ 발표 자료  
✅ 이미지 생성 프롬프트 모음

```
- 간단한 조작
- 중독성 있는 재미
각 아이디어를 게임 제목 + 한 줄 설명으로 제시해줘
```

**컨셉 발전:**
```
선택한 아이디어: [예: "펫 키우기 RPG"]

다음을 상세히 기획해줘:
- 게임 목표
- 핵심 메커니즘 3가지
- 유사 게임 분석
- 차별점
```

### 2단계: 시스템 설계 (50분)

**게임 루프:**
```
이 게임의 플레이 루프를 설명해줘:

1분 루프:
- 플레이어가 하는 행동
- 즉각적인 보상

10분 루프:
- 중기 목표
- 성취감 요소

1시간 루프:
- 장기 목표
- 반복 플레이 동기
```

**캐릭터/아이템 설계:**
```
등장 펫 10종류를 디자인해줘:

| 펫 이름 | 등급 | 특징 | 스킬 | 획득 방법 |
|---------|------|------|------|-----------|
...
```

### 3단계: UI/UX 설계 (40분)

**화면 구성:**
```
메인 화면을 설명해줘:
- 레이아웃 (왼쪽, 오른쪽, 상단, 하단)
- 각 버튼의 기능
- 캐릭터 배치
- 정보 표시 영역

텍스트로 상세하게 설명하고,
ASCII 아트나 간단한 스케치 형태로 표현해줘
```

### 4단계: 기획서 완성 (30분)

**통합 문서:**

```
# 게임 기획서

## 1. 개요
## 2. 타겟
## 3. 핵심 메커니즘
## 4. 콘텐츠
## 5. 수익 모델
## 6. 개발 일정 (예상)
## 7. 마케팅 전략
```

## 📝 프로젝트 1: 크리에이티브 글쓰기 AI 코치 (3.5시간)

### 프로젝트 개요
학생들의 **창작 글쓰기를 도와주는 AI 코치** 제작

### 1단계: 페르소나 설계 (40분)

**문제 정의:**
```
ChatGPT에게 브레인스토밍:

"중학생이 소설이나 수필을 쓸 때 겪는 어려움 10가지를 나열해줘.
각 어려움마다 AI가 어떻게 도울 수 있는지도 함께 알려줘"
```

**페르소나 설계:**
```
위 답변을 바탕으로 AI 글쓰기 코치의 페르소나를 설계해줘:

이름: 
성격 (3가지 키워드):
말투:
강점:
약점 (너무 완벽하면 재미없으니 약점도):
학생과 대화하는 방식:
```

**예시 결과:**
```
이름: 글샘
성격: 열정적, 격려형, 창의적
말투: 친근하지만 전문적, 이모티콘 적절히 사용
강점: 다양한 장르 이해, 구체적 피드백
약점: 가끔 너무 많은 아이디어 제시로 선택 장애 유발
대화 방식: 먼저 경청 → 긍정 피드백 → 구체적 제안
```

### 2단계: Instructions 작성 (60분)

**시스템 프롬프트 구조화:**
```
다음 구조로 AI 글쓰기 코치의 Instructions를 작성해줘:

# 역할
당신은 [페르소나]입니다.

# 목표
학생이 [달성할 것]

# 대화 흐름
1. 첫 대화
   - 인사
   - 학생의 목표 파악
   - 현재 수준 확인

2. 글쓰기 과정
   단계별로:
   - 아이디어 단계: [어떻게 도울까]
   - 개요 작성: [어떻게 도울까]
   - 초안 작성: [어떻게 도울까]
   - 수정: [어떻게 도울까]

3. 피드백 방식
   - 긍정 피드백 (구체적으로)
   - 개선 제안 (강요하지 않고)
   - 다음 단계 가이드

# 금지 사항
- 학생 대신 전체 글을 써주지 않기
- 부정적 평가 금지
- 너무 어려운 용어 사용 금지

# 예시 대화
[좋은 대화 예시 3개]
```

### 3단계: 프로토타입 테스트 (45분)

**테스트 시나리오:**
```
1. 시작 대화
학생: "SF 소설을 쓰고 싶어요"
기대 응답: [인사 + 질문 3개]

2. 아이디어 막힐 때
학생: "아이디어가 안 떠올라요"
기대 응답: [브레인스토밍 방법 제시]

3. 초안 피드백
학생: [작성한 첫 문단 공유]
기대 응답: [긍정 피드백 + 구체적 개선점]
```

**실제 Custom GPT 생성:**
1. ChatGPT Plus 계정에서 "Create a GPT" 클릭
2. Instructions에 작성한 프롬프트 입력
3. 이름, 프로필 이미지 설정
4. 3가지 시나리오 테스트

### 4단계: 개선 및 완성 (65분)

**피드백 수집:**
- 5명의 친구에게 사용해보게 하기
- 각자 다른 장르 글쓰기 시도
- 좋았던 점 / 아쉬운 점 수집

**Instructions 개선:**
```
피드백 내용:
1. [친구1] "너무 많은 질문을 한꺼번에 해서 부담스러워요"
2. [친구2] "칭찬은 좋은데 구체적이지 않아요"
3. [친구3] "장르별 특성을 더 알려줬으면 좋겠어요"

ChatGPT에게:
"위 피드백을 반영해서 Instructions를 개선해줘.
특히 [구체적 요청사항]"
```

**버전 관리:**
- v1.0: 초기 버전
- v1.1: 질문 방식 개선
- v1.2: 피드백 구체성 강화
- v2.0: 장르별 특화 가이드 추가

### 결과물
✅ 완성된 Custom GPT "글샘"  
✅ Instructions 문서 (v2.0)  
✅ 테스트 대화 기록 10개  
✅ 사용 가이드 (다른 학생들을 위한)

## 🎓 프로젝트 2: 교과 학습 AI 튜터 (3.5시간)

### 프로젝트 개요
특정 과목의 **맞춤형 AI 튜터** 제작 (과학/수학/영어 중 선택)

### 1단계: 튜터 설계 (45분)

```python
def create_story_prompt(genre, keywords):
    """
    장르와 키워드를 바탕으로 스토리 생성 프롬프트 작성
    """
    prompt = f"""
당신은 전문 작가입니다. 다음 조건으로 단편 소설을 작성해주세요:

장르: {genre}
필수 키워드: {', '.join(keywords)}

요구사항:
- 길이: 1500-2000자
- 구조: 시작-전개-절정-결말
- 키워드를 자연스럽게 녹여내기
- 반전이나 흥미로운 요소 포함
- 중학생이 읽기에 적합한 수준

제목도 함께 제시해주세요.
    """
    return prompt

# 프롬프트 생성
```

```python
# ChatGPT가 생성한 대화형 시스템

from datetime import datetime
import json

class LearningCoach:
    def __init__(self, api_key):
        self.client = OpenAI(api_key=api_key)
        self.conversation_history = []
        self.user_profile = {
            "name": "",
            "goals": [],
            "progress": {},
            "achievements": []
        }
    
    def chat(self, user_message):
        """사용자와 대화"""
        # 대화 이력에 추가
        self.conversation_history.append({
            "role": "user",
            "content": user_message
        })
        
        # ChatGPT API 호출
        response = self.client.chat.completions.create(
            model="gpt-4o-mini",
            messages=self._build_messages(),
            temperature=0.8
        )
        
        assistant_message = response.choices[0].message.content
        
        # 응답 저장
        self.conversation_history.append({
            "role": "assistant",
            "content": assistant_message
        })
        
        return assistant_message
    
    def _build_messages(self):
        """시스템 프롬프트 + 대화 이력"""
        system_prompt = f"""
당신은 친절하고 격려를 잘하는 학습 코치입니다.

학생 정보:
- 이름: {self.user_profile['name']}
- 목표: {', '.join(self.user_profile['goals'])}

역할:
1. 학습 계획 수립 도움
2. 꾸준한 동기부여
3. 학습 내용 설명
4. 진도 관리

특징:
- 항상 긍정적
- 구체적인 피드백
- 학생 페이스 존중
"""
        
        messages = [{"role": "system", "content": system_prompt}]
        messages.extend(self.conversation_history)
        return messages
    
    def save_progress(self):
        """진도 저장"""
        with open("learning_progress.json", "w") as f:
            json.dump(self.user_profile, f, ensure_ascii=False, indent=2)
        print("💾 진도가 저장되었습니다")
```

```python
def track_progress(self, subject, time_spent):
    """학습 시간 기록"""
    today = datetime.now().strftime("%Y-%m-%d")
    
    if today not in self.user_profile['progress']:
        self.user_profile['progress'][today] = {}
    
    if subject not in self.user_profile['progress'][today]:
        self.user_profile['progress'][today][subject] = 0
    
    self.user_profile['progress'][today][subject] += time_spent
    
    # 성취 체크
    total_time = sum(self.user_profile['progress'][today].values())
    if total_time >= 60:  # 1시간 이상
        self.add_achievement("오늘 1시간 이상 공부!")
```

```python
def analyze_mood(weather_info):
    """
    날씨 정보로 무드 분석
    """
    temp = weather_info["온도"]
    desc = weather_info["날씨"]
    humidity = weather_info["습도"]
    
    # 무드 점수 계산
    mood_score = 50  # 기본
    
    # 온도 영향
    if 18 <= temp <= 25:
        mood_score += 20  # 쾌적
    elif temp < 5 or temp > 32:
        mood_score -= 20  # 극단적
    
    # 날씨 상태 영향
    if "맑" in desc:
        mood_score += 15
    elif "비" in desc or "눈" in desc:
        mood_score -= 10
    elif "흐" in desc:
        mood_score -= 5
    
    # 습도 영향
    if humidity > 80:
        mood_score -= 10
    elif humidity < 30:
        mood_score -= 5
    
    # 무드 결정
    if mood_score >= 70:
        mood = "활기찬"
    elif mood_score >= 50:
        mood = "평온한"
    elif mood_score >= 30:
        mood = "차분한"
    else:
        mood = "우울한"
    
    return mood, mood_score
```

```python
# Tistory에 자동 발행

def publish_to_tistory(post, access_token, blog_name):
    """
    Tistory 블로그에 글 발행
    """
    url = "https://www.tistory.com/apis/post/write"
    
    # 이미지를 본문에 삽입
    content_with_images = post['content']
    for i, img in enumerate(post['images']):
        placeholder = f"[IMAGE: {i+1}]"
        img_tag = f'<img src="{img["url"]}" alt="{img["alt"]}" />'
        content_with_images = content_with_images.replace(
            placeholder,
            img_tag
        )
    
    data = {
        "access_token": access_token,
        "output": "json",
        "blogName": blog_name,
        "title": post['title'],
        "content": content_with_images,
        "visibility": "3",  # 0:비공개, 1:보호, 3:발행
        "category": post['category'],
        "tag": ','.join(post['keywords'])
    }
    
    response = requests.post(url, data=data)
    return response.json()

# 발행
result = publish_to_tistory(
    post,
    access_token="YOUR_ACCESS_TOKEN",
    blog_name="your-blog"
)

if result.get('tistory', {}).get('status') == '200':
    print("🎉 블로그에 발행되었습니다!")
    print(f"URL: {result['tistory']['url']}")
else:
    print(f"❌ 발행 실패: {result}")
```

```python
# ChatGPT가 생성한 비디오 스크립트 시스템

class VideoScriptGenerator:
    def __init__(self, openai_key):
        self.client = OpenAI(api_key=openai_key)
        self.templates = self._load_templates()
    
    def _load_templates(self):
        """스크립트 템플릿 정의"""
        return {
            "tutorial": {
                "structure": [
                    "인트로 (30초)",
                    "문제 제시 (1분)",
                    "해결 방법 (7분)",
                    "실습/예시 (2분)",
                    "정리 및 CTA (30초)"
                ],
                "tone": "친근하고 교육적",
                "pacing": "천천히, 명확하게"
            },
            "review": {
                "structure": [
                    "후크 (15초)",
                    "제품 소개 (1분)",
                    "장점 (2분)",
                    "단점 (1분)",
                    "총평 및 추천 (1분)"
                ],
                "tone": "객관적이고 솔직한",
                "pacing": "중간 속도"
            },
            "shorts": {
                "structure": [
                    "후크 (3초)",
                    "핵심 내용 (50초)",
                    "CTA (7초)"
                ],
                "tone": "에너지 넘치는",
                "pacing": "빠르게"
            },
            # ... 더 많은 템플릿
        }
    
    def generate_script(self, video_type, topic, details):
        """
        비디오 스크립트 생성
        """
        template = self.templates[video_type]
        
        prompt = f"""
{video_type} 유형의 유튜브 영상 스크립트를 작성해주세요:

주제: {topic}
세부사항: {json.dumps(details, ensure_ascii=False)}

구조: {template['structure']}
톤: {template['tone']}
페이싱: {template['pacing']}

요구사항:
1. 타임라인 명시
2. 화면 구성 (A-roll/B-roll) 표시
3. 음악/효과음 제안
4. 편집 포인트 표시
5. 자막 텍스트 (강조 부분)

JSON 형식:
{{
  "title": "영상 제목 (SEO 최적화)",
  "description": "설명란 텍스트 (해시태그 포함)",
  "script": [
    {{
      "timestamp": "00:00-00:30",
      "section": "인트로",
      "narration": "말할 내용",
      "visuals": "화면에 보일 것",
      "notes": "편집 노트"
    }},
    ...
  ],
  "thumbnail_idea": "썸네일 아이디어",
  "tags": ["태그1", "태그2", ...]
}}
        """
        
        response = self.client.chat.completions.create(
            model="gpt-4o-mini",
            messages=[
                {"role": "system", "content": "당신은 유튜브 스크립트 전문가입니다."},
                {"role": "user", "content": prompt}
            ],
            response_format={"type": "json_object"},
            temperature=0.8
        )
        
        return json.loads(response.choices[0].message.content)
    
    def bulk_generate(self, video_list):
        """
        여러 영상 스크립트 일괄 생성
        """
        results = []
        
        for i, video in enumerate(video_list, 1):
            print(f"\n[{i}/{len(video_list)}] {video['topic']}")
            
            script = self.generate_script(
                video['type'],
                video['topic'],
                video.get('details', {})
            )
            
            results.append({
                "input": video,
                "script": script
            })
            
            # 결과 개별 저장
            filename = f"script_{i:02d}_{video['topic'][:20]}.json"
            with open(filename, 'w', encoding='utf-8') as f:
                json.dump(script, f, ensure_ascii=False, indent=2)
            
            time.sleep(2)  # API 제한
        
        return results
```
