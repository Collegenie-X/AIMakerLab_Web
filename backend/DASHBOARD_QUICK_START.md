# 🚀 대시보드 5분 빠른 시작

## ✅ 완료된 작업

Django Admin에 **통합 대시보드**가 추가되었습니다!

### 🎯 주요 기능
- 📊 **통합 대시보드** - 전체 CRUD 현황 한눈에 보기
- 📅 **일별 통계** - 최근 활동 추이 (7일~90일)
- 📆 **월별 통계** - 트렌드 분석 (3~24개월)
- 📋 **모델별 현황** - 27개 모델 데이터 요약
- 🔔 **최근 활동** - 실시간 로그

---

## 🚀 빠른 시작 (5분)

### 1단계: 서버 실행
```bash
cd backend
python manage.py runserver
```

### 2단계: Admin 접속
```
http://localhost:8000/admin/
```

### 3단계: 대시보드 확인
로그인하면 자동으로 **통합 대시보드** 화면이 나타납니다!

## 📈 통계 페이지

### 일별 통계
- **URL**: `/admin/dashboard/daily/`
- **기간 선택**: 7일 / 14일 / 30일 / 60일 / 90일
- **차트**: Line Chart
- **테이블**: 날짜별 상세 데이터

### 월별 통계
- **URL**: `/admin/dashboard/monthly/`
- **기간 선택**: 3개월 / 6개월 / 12개월 / 24개월
- **차트**: Line Chart + Bar Chart
- **테이블**: 전월 대비 증감

### 상세 통계
- **URL**: `/admin/dashboard/stats/`
- **구성**: 앱별 상세 통계 (5개 섹션)
  - 사용자 통계 (6개 지표)
  - 문의 통계 (9개 지표)
  - 제품 통계 (10개 지표)
  - 갤러리 통계 (6개 지표)
  - 커리큘럼 통계 (7개 지표)

## 📋 통계 항목

### 핵심 지표 (6개 카드)
1. **👥 총 사용자** - 전체/활성/인증/오늘 가입
2. **📞 총 문의** - 전체/상태별/오늘 문의
3. **📦 제품 & 영상** - 제품/리뷰/영상/평균 평점
4. **🎨 갤러리** - 작품/후기/조회수/좋아요
5. **📚 커리큘럼** - 과정/프로젝트/모듈/자료
6. **📅 일정** - 전체/가능/만석

### 모델별 데이터 (27개)
- Accounts: User
- Inquiry: Inquiry, Schedule
- Products: Product, Review, Video, Photo
- Gallery: GalleryItem
- Curriculum: Curriculum, Project, Module, Material
- Home: HeroSlide, Feature, Highlight

## 🔧 기술 스택

### 백엔드
- **Django 5.0.1** - Admin 커스터마이징
- **Python ORM** - 통계 쿼리 (Count, Sum, Avg)

### 프론트엔드
- **Django Templates** - HTML 렌더링
- **Chart.js 4.4.0** - 차트 라이브러리
- **CSS Grid** - 반응형 레이아웃

### 아키텍처
```
CustomAdminSite (custom_admin.py)
    ↓
DashboardStats (admin_dashboard.py)
    ↓
Templates (templates/admin/)
    ↓
Chart.js (CDN)
```
